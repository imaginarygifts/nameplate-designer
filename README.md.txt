# Nameplate Designer

A free, browser-based nameplate and product design tool.

## Features
- Add text
- Choose custom fonts
- Select shapes
- Add logos and borders
- Move, resize, rotate elements
- Export final design as JPG
- Works offline
- No plugins
- No backend

## How to Use
1. Download or clone this repository
2. Open `index.html` in any modern browser
3. Start designing
4. Click "Save as JPG" to download your design
